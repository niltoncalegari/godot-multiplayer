# Godot SQLite Plugin

Este projeto usa o plugin [godot-sqlite](https://github.com/2shady4u/godot-sqlite) para suporte a SQLite.

## Instalação

1. Baixe o plugin de: https://github.com/2shady4u/godot-sqlite
2. Coloque os arquivos na pasta `addons/godot-sqlite/`
3. Ative o plugin nas configurações do projeto (Project > Project Settings > Plugins)

## Uso

O plugin fornece a classe `SQLite` que pode ser usada para operações de banco de dados.

## Nota

Se o plugin não estiver disponível, o sistema pode funcionar com arquivos JSON como fallback (implementação futura).
