# Diretório para dados do servidor (SQLite)

Este diretório contém o banco de dados SQLite do servidor.

## Localização do Banco

O banco será criado em: `server_data/game_database.db`

## Conectar no DBeaver

1. Abra o DBeaver
2. Crie nova conexão SQLite
3. No campo "Path", navegue até: `[projeto]/server_data/game_database.db`
4. Teste a conexão e conecte

## Nota

O banco será criado automaticamente quando o servidor iniciar pela primeira vez.
